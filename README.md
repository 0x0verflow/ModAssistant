# ModAssistant Chinese Version
简单的Beat Saber MOD安装器，类似于 [Beat Saber Mod Manager](https://github.com/beat-saber-modding-group/BeatSaberModInstaller)

# [下载地址](https://github.com/Assistant/ModAssistant/releases/latest)

具有以下功能：
* 依赖解析
* 检测已安装的MOD
* 移除MOD
* 一键安装支持
